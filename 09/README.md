## 🔑 Основные команды Alembic

| Описание | Команда |
|----------|---------|
| Создать новую миграцию | `alembic revision --autogenerate -m "описание изменений"` |
| Применить все миграции | `alembic upgrade head` |
| Откатить на одну миграцию | `alembic downgrade -1` |
| Откатить к конкретной версии | `alembic downgrade <version_id>` |
| Показать историю миграций | `alembic history` |
| Показать текущую версию | `alembic current` |

## 📦 Зависимости

- **fastapi** — веб-фреймворк
- **sqlalchemy** — ORM для работы с БД
- **alembic** — инструмент миграции БД
- **uvicorn** — ASGI сервер

## 🔧 Настройка

### Для SQLite (по умолчанию)
В файле `alembic.ini`:
```ini
sqlalchemy.url = sqlite:///./products.db
```

### Для PostgreSQL
Замените на:
```ini
sqlalchemy.url = postgresql://user:password@localhost:5432/dbname
```
