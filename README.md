## 📋 Описание

Данный репозиторий содержит решения контрольных работ №1 и №2 по дисциплине «Технологии разработки серверных приложений». Проект реализован на фреймворке **FastAPI** и демонстрирует навыки разработки серверных приложений, работы с моделями данных, валидацией, аутентификацией на основе cookie и работой с HTTP-заголовками.

### 🛠 Технологический стек
- **Язык:** Python 3.10+
- **Фреймворк:** FastAPI
- **Сервер:** Uvicorn
- **Валидация данных:** Pydantic (v2)
- **Безопасность:** itsdangerous, UUID, Cookies (HttpOnly)
- **Тестирование:** Curl, Postman, Browser
